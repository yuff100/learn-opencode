## 核心思路

### AIDA 模型

| 阶段 | 英文 | 说明 | 文案作用 |
|-----|------|------|---------|
| A | Attention | 注意 | 标题抓眼球 |
| I | Interest | 兴趣 | 痛点引共鸣 |
| D | Desire | 欲望 | 好处激渴望 |
| A | Action | 行动 | CTA 促转化 |

### 文案类型速览

| 类型 | 特点 | 字数 |
|-----|------|------|
| 朋友圈文案 | 短平快、配图 | 50-150 字 |
| 公众号推文 | 故事化、深度 | 1000-3000 字 |
| 产品详情页 | 结构化、卖点突出 | 500-1500 字 |
| 广告语 | 极简、有记忆点 | 5-20 字 |

## 跟我做

### 第 1 步：分析产品卖点

**为什么**  
好文案从理解产品开始。

切换到 Plan 模式：

```
我要推广一款「智能降噪耳机」，售价 599 元。
帮我分析：
1. 目标用户画像（年龄、职业、痛点）
2. 核心卖点提炼（3 个）
3. 与竞品的差异化优势
4. 可能的购买顾虑及如何打消
```

### 第 2 步：生成标题党标题

**为什么**  
标题决定 80% 的点击率。

```
为这款耳机生成 10 个标题，要求：
1. 使用数字（如"3 个理由"）
2. 制造好奇（如"原来..."）
3. 痛点切入（如"还在被噪音困扰？"）
4. 结果导向（如"效率提升 50%"）

每种类型至少 2-3 个
```

### 第 3 步：写 AIDA 结构文案

<AdInArticle />

**为什么**  
结构化文案更有说服力。

切换到 Build 模式：

```
用 AIDA 模型写一篇朋友圈文案（150 字以内）：

【Attention】开头一句话抓注意力
【Interest】描述用户痛点
【Desire】展示产品如何解决
【Action】促销信息 + 行动号召

保存为 copy/朋友圈文案.md
```

### 第 4 步：生成 A/B 变体

**为什么**  
多版本测试找出最优。

```
基于刚才的文案，生成 3 个变体版本：
- 版本 A：强调「安静」体验
- 版本 B：强调「性价比」
- 版本 C：强调「专业人士都在用」

保存为 copy/AB测试版本.md
```

### 第 5 步：完整文案套件

**为什么**  
一次生成多渠道素材。

```
为这款耳机生成完整文案套件：
1. 广告语（10 字以内）
2. 朋友圈文案（150 字）
3. 小红书笔记（300 字 + 5 个话题标签）
4. 产品详情页文案（800 字，含卖点分段）

全部保存到 copy/ 目录下
```

## 📋 魔法 Prompt

### 📢 AIDA 文案生成
> 预期效果：生成结构化营销文案，提升转化率

```
## 角色
你是资深文案策划，擅长用 AIDA 模型写转化率高的营销文案。

## 任务
用 AIDA 模型为产品写营销文案。

## 输入信息
### 必填项
- 产品名称：【名称】
- 核心卖点：【1-3 个卖点】
- 目标用户：【用户画像】

### 选填项
- 文案场景：【朋友圈/公众号/详情页?】（默认：朋友圈）
- 字数要求：【字数?】（默认：150字）
- 促销信息：【限时/限量/折扣?】

## 输出规范
按 AIDA 四阶段结构输出：

### A - Attention（吸引注意）
> 一句话抓眼球（可用提问/数字/反差）

【开场文案】

### I - Interest（引发兴趣）
> 描述目标用户的痛点场景，引发共鸣

【痛点描述】

### D - Desire（激发欲望）
> 展示产品如何解决痛点 + 独特优势

【产品价值】

### A - Action（促进行动）
> 促销信息 + 明确的行动号召（限时/限量/专属）

【CTA 文案】

---

**完整文案**：
【整合后的完整文案】

## 约束条件
- ✅ 每个阶段功能明确
- ✅ 语言口语化，避免书面语
- ✅ CTA 要具体，有紧迫感
- ❌ 避免夸大宣传、违反广告法
- ❌ 避免使用"最""第一"等绝对化用语
```

## 踩坑提醒

| 现象 | 原因 | 解决 |
|-----|-----|-----|
| 文案太平淡 | 没用 AIDA 结构 | 按 AIDA 四个部分分别写 |
| 标题不够吸引人 | 只生成一个版本 | 要求生成 10 个备选 |
| 没突出产品卖点 | 没分析产品 | 先做卖点分析再写文案 |
