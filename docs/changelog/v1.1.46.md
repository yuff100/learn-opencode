# v1.1.46：修复 Think Blocks 显示问题

**发布日期**：2026年01月30日

---

## 一句话总结

这次更新修复了 Think Blocks（思考过程）无法正常显示的问题，让你能看到 AI 的完整思考过程。

---

## 问题修复：Think Blocks 显示问题

### 之前的问题

使用支持 Thinking 模式的模型（如 Claude 3.5 Sonnet）时，AI 的思考过程（Think Blocks）无法正常显示在对话中。这是因为 AI SDK 中间件阻止了 think blocks 作为助手消息内容返回。

### 现在的情况

移除了导致问题的 AI SDK 中间件，Think Blocks 现在可以正常显示了。

### 有什么好处？

- 可以看到 AI 的完整思考过程
- 更好地理解 AI 是如何得出结论的
- 对于调试和学习非常有帮助
