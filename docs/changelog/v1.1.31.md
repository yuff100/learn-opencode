# v1.1.31：子 Agent 不占额度，新增复古主题

**发布日期**：2026年01月22日

---

## 一句话总结

这次更新带来了实打实的福利：子 Agent 的调用不再计入你的额度限制！同时增加了复古风格的 Gruvbox 主题和繁体中文支持。

---

## 重磅福利：子 Agent 免额度

### 这次更新了什么？

现在，由主 Agent 启动的**子 Agent 会话**（Subagent sessions），将被标记为 `agent-initiated`，不再计入你的 API 使用额度限制（Quota Limits）。

### 这意味着什么？

以前你可能担心叫几个帮手（子 Agent）会消耗太多额度。现在：
- 你可以放心地让 AI 启动 `search` agent 去查资料。
- 或者是启动 `code-review` agent 来审查代码。
- 这些子任务的消耗将更加“宽容”（具体取决于你的额度策略，但通常意味着不占主额度或有单独配额）。

### 如何享受？

无需任何操作，升级到 v1.1.31 即可生效。

---

## 新外观：Gruvbox 主题

### 什么是 Gruvbox？

Gruvbox 是一款非常经典的复古风格配色主题，以暖色调为主，对比度适中，长时间看代码不累眼。

### 如何切换？

在 OpenCode Desktop 或 Web 版的设置中：
1. 打开 **Settings** (设置)
2. 找到 **Theme** (主题)
3. 选择 **Gruvbox**

---

## 语言支持：繁体中文

### 新增支持

OpenCode 现在支持**繁体中文**界面了！同时，原来的 "Chinese" 选项已更名为 "Chinese (Simplified)"（简体中文），以便区分。

### 如何设置？

1. 打开 **Settings**
2. 找到 **Language**
3. 选择 **Traditional Chinese**

---

## 要不要升级？

| 情况 | 建议 |
|------|------|
| 经常使用 Task/Subagent 功能 | ✅ **必须升级** (省额度！) |
| 喜欢复古配色 | ✅ 推荐升级 |
| 习惯繁体中文界面 | ✅ 推荐升级 |

**升级命令**：

```bash
opencode upgrade
```
