# v1.1.29：多语言支持与 PHP 格式化

**发布日期**：2026年01月21日

---

## 一句话总结

OpenCode 开始支持多语言（国际化），新增 PHP Laravel Pint 格式化器，Windows 窗口控制更自然。

---

## 新功能：多语言支持

### 这次更新了什么？
OpenCode 桌面应用开始支持多语言界面！
- 新增了日语、丹麦语、法语、西班牙语、德语、韩语的翻译。
- 会话标题现在会根据你的提问语言自动生成（比如你用中文问，标题就是中文）。

---

## 新功能：PHP Laravel Pint

### 这次更新了什么？
新增了对 PHP `Laravel Pint` 格式化器的支持。

### 如何使用？
如果你的项目中有 `.php` 文件，OpenCode 现在可以使用 Laravel Pint 来自动格式化代码，保持代码风格统一。

---

## 改进：Windows 窗口控制

### 之前的问题
Windows 版本的窗口控制按钮（最小化、最大化、关闭）可能与系统原生体验不太一致。

### 现在的情况
优化了 Windows 版本的窗口控制集成，现在操作更加自然顺滑。

---

## 改进：环境变量控制

### 这次更新了什么？
新增了两个环境变量，给高级用户更多控制权：
- `OPENCODE_DISABLE_PROJECT_CONFIG`：禁用项目配置加载。
- `OPENCODE_DISABLE_FILETIME_CHECK`：跳过文件修改时间检查（对某些特殊文件系统有用）。

---

## 要不要升级？

| 情况 | 建议 |
|------|------|
| PHP 开发者 | ✅ 推荐，新增格式化支持 |
| Windows 用户 | ✅ 推荐，窗口体验更好 |
| 需要多语言 | ✅ 推荐 |

**升级命令**：

```bash

```
