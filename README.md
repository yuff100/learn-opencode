## 📚 课程内容

```
第一阶段：快速起步（必修 · 30 分钟）
    ↓   安装、网络配置、连接模型、第一次对话
第二阶段：日常使用（必修 · 1.5 小时）
    ↓   界面操作、会话管理、文件引用
第三阶段：高效工作流（必修 · 1 小时）
    ↓   Plan/Build 模式、Agent 协作、项目初始化
第四阶段：场景实战（选修 · 选一条线）
    │
    ├── ✍️  内容创作线：公众号、小红书、小说、剧本、营销文案
    ├── 💻 程序员线：日常开发、重构测试、文档 Git、CI/CD
    └── 📊 效率办公线：文件整理、数据处理、自动化脚本
    ↓
第五阶段：深度定制（选修 · 按需）
        自定义 Agent、Skill、命令、MCP 扩展、IDE 集成
```

## 🚀 立即开始

### 在线阅读（推荐）

👉 **[learnopencode.com](https://learnopencode.com)**

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/vbgate/learn-opencode.git
cd learn-opencode

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
